Harcharan Singh Memorial Public Senior Secondary School - Official Website

A modern, professional website showcasing our institution's excellence in education, state-of-the-art facilities, and commitment to holistic student development.

## Features

- **3D Hero Section** - Immersive particle animations and geometric shapes
- **Interactive Gallery** - Filterable photo gallery with smooth transitions
- **Smart Classroom Showcase** - 3D animated laptop highlighting digital learning
- **Responsive Design** - Optimized for all devices (desktop, tablet, mobile)
- **Smooth Animations** - Scroll-triggered fade-in effects throughout
- **Contact Form** - Easy inquiry submission for prospective students
- **Google Maps Integration** - Interactive location map with directions

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with animations and gradients
- **JavaScript (ES6)** - Interactive functionality
- **Three.js** - 3D graphics and animations
- **Google Fonts** - Cormorant Garamond & Outfit typography
- **Google Maps API** - Location embedding
